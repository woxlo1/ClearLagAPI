# ClearLagAPI

ClearLag によって削除されたアイテムをプレイヤーが後から復元できる Paper / Spigot プラグインです。

## 機能

* ClearLag によって削除されたアイテムを自動保存
* プレイヤーごとに削除アイテムを管理
* GUIからアイテムを復元可能
* 復元済みアイテムは一覧から自動削除
* アイテム受取期限（7日）
* 削除日時を表示
* アイテム所有者を記録
* データをYAMLで保存

## 動作環境

* Java 17+
* Paper 1.20.4
* ClearLag
* OraPluginAPI

## コマンド

| コマンド         | 説明             |
| ------------ | -------------- |
| `/cla items` | 削除されたアイテム一覧を開く |

## GUI

削除されたアイテム一覧では以下の情報が表示されます。

* 削除日時
* 落としたプレイヤー
* 復元可能状態

アイテムをクリックするとインベントリへ復元されます。

## アイテム保存の流れ

1. プレイヤーがアイテムをドロップ
2. アイテムに所有者情報を付与
3. ClearLagがアイテムを削除
4. 削除時にデータを保存
5. プレイヤーがGUIから復元

## 受取期限

削除されたアイテムは7日間保存されます。

期限を超えたアイテムは自動的に削除されます。

## データ保存

保存先:

```text
plugins/ClearLagAPI/data.yml
```

保存内容:

* 所有者UUID
* 所有者名
* アイテムデータ
* 削除日時
* 復元状態
