# ClearLagAPI

ClearLag によって削除されたアイテムをプレイヤーが後から復元できる Paper / Spigot プラグインです。

## 機能

* ClearLag によって削除されたアイテムを自動保存
* プレイヤーごとに削除アイテムを管理
* GUI からアイテムを復元可能
* 復元済みアイテムは一覧から自動削除
* アイテム受取期限（7日）
* 削除日時・残り期限（日/時間/分）を表示
* アイテム所有者を記録
* データを YAML で保存
* OP 専用の管理コマンド・GUI

## 動作環境

* Java 17+
* Paper 1.20.4
* ClearLag
* OraPluginAPI

## コマンド

### 一般プレイヤー

| コマンド        | 説明 |
|-------------| --- |
| `/cla list` | 削除されたアイテム一覧を開く |
| `/cla help` | ヘルプを表示する |

### OP 専用

| コマンド | 説明 |
| --- | --- |
| `/cla op` | 全プレイヤーの削除アイテムを閲覧する |
| `/cla op <プレイヤー名>` | 特定プレイヤーの削除アイテムを閲覧する |

エイリアス: `/clearlagapi`

## Tab 補完

* 第 1 引数: サブコマンドを補完（OP には `op` も表示）
* 第 2 引数: `/cla op` 入力時にオンラインプレイヤー名を補完

## GUI

### アイテム一覧（`/cla list`）

削除されたアイテム一覧では以下の情報が表示されます。

* 削除日時
* 落としたプレイヤー
* 残り受取期限（日・時間・分）

アイテムをクリックするとインベントリへ復元されます。

### 管理一覧（`/cla op`）OP 専用

* 全プレイヤーの未復元アイテムを一覧表示
* プレイヤー名・削除日時・残り期限を表示
* 閲覧専用（復元操作なし）

## アイテム保存の流れ

1. プレイヤーがアイテムをドロップ
2. アイテムに所有者情報を付与
3. ClearLag がアイテムを削除
4. 削除時にデータを保存
5. プレイヤーが GUI から復元

## 受取期限

削除されたアイテムは 7 日間保存されます。

期限を超えたアイテムは自動的に削除されます。

## パーミッション

| パーミッション           | 説明                   | デフォルト |
|-------------------|----------------------| --- |
| `clearlagapi.use` | `/cla` コマンドを使用できる    | 全員 |
| `clearlagapi.op`  | `/cla op` コマンドを使用できる | OP のみ |

## データ保存

保存先:

```
plugins/ClearLagAPI/data.yml
```

保存内容:

* 所有者 UUID
* 所有者名
* アイテムデータ
* 削除日時
* 復元状態
